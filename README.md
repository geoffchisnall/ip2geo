# ip2geo

##### Summary

- lookup to take an IP or Domain Name and see where it's Geo Location is.  
it is using the https://ipwhois.io API. - Limited to 10000 searches per month.  
-usage  
./ip2geo.py ipaddress")  
example - ./ip2geo.py 8.8.8.8")  
example - ./ip2geo.py www.google.com")  
  
--------------------  
  
#### requirements:  
- python3  
- jq
